Web2PG Extension Icons
=======================

This directory should contain the following icon files:

- icon16.png  (16x16 pixels)
- icon48.png  (48x48 pixels)
- icon128.png (128x128 pixels)

You can create these icons using:
1. An online icon generator (e.g., favicon.io, canva.com)
2. Image editing software (Photoshop, GIMP, etc.)
3. AI image generators

Recommended icon design:
- A simple database or archive symbol
- Blue color scheme (#2196F3) to match the UI
- Clean, minimalist design

For now, the extension will work without icons but will show the default Chrome extension icon.

Temporary solution:
You can use any PNG images and resize them to the required dimensions.
